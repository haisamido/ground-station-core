# How to use

## Dependencies

- `gnu make`
- `bash`

### Optional but very useful

- `task` a contemporary version of `make`
- `lazydocker`

Which can be installed via `make install-lazydocker install-taskfile`

## prep

```bash
make install-lazydocker install-taskfile
make env-create

```

## run

```bash
make

```
